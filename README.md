# Superstore-Retail-Performance-Dashboard
• Developed an interactive Superstore Retail Performance Dashboard in Power BI, analyzing 117K+ sales, 19K profit, and 460 returns
across 47 cities.
• Built dynamic visuals (KPIs, bar charts, maps) using Power BI, DAX, and Power Query to enable data-driven decision-making.
• Used Power Query to clean and transform raw datasets from multiple sources, reducing data refresh time and ensuring 100%
accuracy across all regional reporting
