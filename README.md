# Superstore Retail Performance Dashboard 📊

## 📌 Project Overview
This project focuses on building an interactive, end-to-end Power BI dashboard designed to analyze and evaluate the operational and financial performance of a global retail superstore. By translating raw transactional data into dynamic visuals, the dashboard empowers stakeholders to track profitability drivers, monitor geographic growth, and pinpoint inefficiencies across product categories and shipping logistics.

## 🛠️ Core Features & Capabilities
* **Dynamic Sales & Profit Analysis:** Interactive KPIs tracking total revenue, net profit margins, and order volumes across dynamic timeframes.
* **Geographic Mapping:** Visualized sales distribution across regions to identify top-performing markets and localized revenue deficits.
* **Product Segment Deep-Dive:** Clean breakdowns of product categories (Furniture, Office Supplies, Technology) to reveal hidden margin killers.
* **Advanced Data Modeling:** Developed robust DAX (Data Analysis Expressions) measures for advanced time-intelligence calculations (YoY Growth, Year-to-Date totals, and profit ratios).
* **ETL & Data Cleaning:** Leveraged Power Query to handle missing data, establish clean star-schema relationships, and format parameters for seamless slicing.

## 📈 Business Impact
Instead of just viewing static lists, decision-makers can use this dashboard to instantly discover which specific product sub-categories are losing money, which regions have the longest shipping delays, and where to allocate marketing budgets next quarter.
